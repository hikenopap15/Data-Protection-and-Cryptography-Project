This paper investigates the mathematical foundations of the cipher with a great focus on the role of the S-box in ensuring non-linearity, the permutation layer's function in achieving diffusion, 
and the significance of modular arithmetic and bitwise operations in the key scheduling process. 
A step-by-step one-round encryption example is presented using the official S-box and permutation functions to depict how the cipher transforms plaintext into ciphertext.
